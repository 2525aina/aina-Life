# 109_タスクボタン文字色変更実装手順

このドキュメントは、ホーム画面のタスクボタンに背景色だけでなく文字色も設定できるようにする実装手順を説明します。

---

## 1. 概要

タスクボタンの視認性とカスタマイズ性を向上させるため、タスクに設定された背景色に加えて、文字色も動的に反映されるように機能を追加します。

---

## 2. 変更内容

### 2.1. `src/hooks/useLogbook.ts` の `Task` インターフェースに `textColor` プロパティを追加

<details>
<summary>クリックしてコードを表示</summary>

タスクデータに文字色情報を保持できるように、`Task`インターフェースに`textColor`プロパティを追加しました。

```typescript
// src/hooks/useLogbook.ts (抜粋)
export interface Task {
  id: string;
  name: string;
  color: string;
  order: number;
  textColor?: string; // ★追加: 文字色プロパティ
}
```
</details>

### 2.2. `src/components/TaskSelector.tsx` で文字色を適用

<details>
<summary>クリックしてコードを表示</summary>

タスクボタンのレンダリング時に、`Task`インターフェースに追加された`textColor`プロパティを使用して、ボタンの文字色を動的に設定するように変更しました。`textColor`が指定されていない場合はデフォルトで白が適用されます。

```typescript
// src/components/TaskSelector.tsx (抜粋)
        <button
          // ... (省略) ...
          style={{ backgroundColor: task.color, color: task.textColor || 'white' }} // ★修正: 文字色を適用
        >
          {task.name}
        </button>
```
</details>

### 2.3. `src/components/TaskFormModal.tsx` に `textColor` 入力フィールドを追加

<details>
<summary>クリックしてコードを表示</summary>

タスク管理画面のタスク編集/追加モーダルに、文字色を設定するための入力フィールドを追加しました。これにより、ユーザーがタスクごとに文字色を自由に設定できるようになります。また、`addTask`および`updateTask`関数を呼び出す際に、この`textColor`がタスクデータとして渡されるように修正しました。

```typescript

    if (isOpen) {
      if (taskToEdit) {
        // ... (省略) ...
        setTextColor(taskToEdit.textColor || '#FFFFFF'); // ★追加: textColorの初期化
      } else {
        // ... (省略) ...
        setTextColor('#FFFFFF'); // ★追加: textColorのリセット
      }
    }
  }, [isOpen, taskToEdit]);

  const handleSubmit = async () => {
    // ... (省略) ...
      if (taskToEdit) {
        await updateTask(taskToEdit.id, { name, color, textColor }); // ★修正: textColorを渡す
      } else {
        await addTask({ name, color, textColor, order: newOrder }); // ★修正: textColorを渡す
      }
    // ... (省略) ...
  };

  return (
    // ... (省略) ...
        {/* ★追加: 文字色入力フィールド */}
        <div className="mb-6">
          <label htmlFor="taskTextColor" className="block text-gray-700 text-sm font-bold mb-2">文字色 (HEXコード)</label>

            type="color"

            value={textColor}

            type="text"
```

## 3. 確認手順

<details>
<summary>確認手順を表示</summary>

1.  アプリケーションを起動し、タスク管理画面に移動します。

4.  ホーム画面に戻り、設定したタスクボタンの背景色と文字色が正しく反映されていることを確認します。
